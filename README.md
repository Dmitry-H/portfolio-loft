Данный сайт является выпускным проектом в рамках курса loftscool "продвинутый курс по веб-разработке" и представляет собой портфолио веб-разработчика.

Автор: Хайс Дмитрий  
Наставник: Андрей Иващенко

Для сборки проекта необходимо выполнить следующие консольные команды:  
git clone https://github.com/Dmitry-H/portfolio-loft.git  
cd portfolio-loft  
npm install  
npm run build

Для запуска live-сервера необходимо выполнить следующую комманду:  
npm start
